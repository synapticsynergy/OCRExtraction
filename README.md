## Usage

>To create a utility to drag and drop images, send them to a server for OCR, then return the extracted text in a CSV file.

## Requirements

- Node 10.2.1 or higher
- NPM 5.6.0 or higher

### Installing Dependencies

From within the root directory:

```sh
cd client
yarn
```
or 
```sh
cd client
npm install
```

## Development
From within the root directory:

```sh
cd client
yarn start
```
or 
```sh
cd client
npm start
```
